# Scholar

論文メモ

    読もうと思っている論文を Issue に上げる
    読んだ論文の要旨を、ファイルに纏める

Format

    # [Title](URL)
    ###著者
    ###会議名

    ## 概要

    ## 先行研究と比べてここがすごい

    ## 手法のキモ

    ## 有効性の評価
    ###データセット
    ###評価尺度
    
 
    ## その他の議論

    ## 次に読むべき論文

    ## コメント
